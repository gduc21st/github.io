A showcase of interactive works, coding challenges, and digital design.

1. Final Project: The True Cost of Your Closet

Interactive Web App (HTML/CSS/JavaScript)

Description: An interactive global/societal impact calculator designed to confront college students with the environmental footprint of fast fashion and textile waste.

How I Solved the Challenge:

I structured a single-page architecture using dynamic CSS display properties to toggle between quiz questions without requiring page reloads. I utilized JavaScript event listeners to capture user selections, store state variables, dynamically calculate a mathematical impact score, and render personalized feedback tailored to the target persona.

2. Assignment 1: Responsive Layout Profile

Layout & Typography (HTML/CSS)

Description: A clean, multi-column semantic profile layout designed to display cleanly across mobile, tablet, and desktop viewports.

How I Solved the Challenge:

I implemented modern CSS Flexbox and Media Queries to handle fluid structural shifts. I solved viewport constraint issues by adopting a mobile-first design strategy, scaling element sizing smoothly using relative EM and REM units instead of absolute pixels.

3. Assignment 2: Interactive Data UI

Dynamic Logic (HTML/CSS/JavaScript)

Description: A front-end interface featuring live input monitoring, data processing, and instantaneous DOM styling adjustments based on user input values.

How I Solved the Challenge:

The core hurdle was ensuring data safety and real-time responsiveness. I bound conditional event-handler logic to JavaScript input events, sanitized user entries using basic parsing variables, and immediately reflected output changes directly into the document structure via innerText manipulation.

4. Assignment 3: Accessible Form Validation

Form Architecture (HTML/CSS/JS)

Description: A interactive multi-field web form equipped with customized client-side verification and visual accessibility queues.

How I Solved the Challenge:

I circumvented standard browser validation errors by overriding native forms with custom JavaScript regular expression tests. I designed contextual error elements that cleanly toggle visibility and adjust color styling dynamically to maintain access guidelines for vision-impaired users.

5. Assignment 4: CSS Animation Workshop

State Transitions & Visual Effects (CSS)

Description: A visual-first experimental assignment exploring asynchronous web triggers, keyframe paths, and state transitions.

How I Solved the Challenge:

I achieved multi-step animations using pure CSS `@keyframes` properties linked to active UI hover tags. By fine-tuning the `transition-timing-function` bezier curves, I produced a natural, hardware-accelerated movement experience without depending on heavyweight external scripting frameworks.
